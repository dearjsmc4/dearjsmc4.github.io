Installing Hexo

헥소 설치

https://hexo.io/docs/
헥소 사이트로 이동하여 시키는 대로 하다보면 되던데
기억을 더듬어서 순서를 적어보자…

갑자기 망할 놈(…) 소리를 들었는데
이 망할 놈이 바로 Git 이다. 번역기가 잘못했네
헥소 설치 전에 Node.js와 Git 을 설치해야 한다.

$ hexo init 폴더명
폴더를 만들고
$ cd 폴더명
그 폴더로 이동하여
$ npm install
설치

앞으로 글을 쓸 때 내가 init 한 그 폴더에서 작업하면 된다.

글을 올릴 때는
$ hexo deploy
명령을 쓰면 되는데 그 전에 내가 글을 올릴 곳을 설정해줘야한다.

나는 깃을 쓰니까
$ npm install hexo-deployer-git –save
hexo-deployer-git 을 설치해야한다.
그리고 _config.yml 파일을 열면 설정을 수정할 수 있다.

몇 가지 커맨드로 간단히 글을 포스팅할 수 있다.

$ hexo new post 포스트제목
포스트제목에 만약 공백이 있으면 “” 로 묶어줘야한다.

$ hexo generate
(정적 파일을 생성)
$ hexo server
(서버를 확인)
$ hexo deploy
(글을 업로드)

정리하다보니 이게 정말 간단한건지…
네이버 블로그에 익숙한 내게는 조금 불편하긴 하다.
아직 익숙하지 않아서 그럴지도.
