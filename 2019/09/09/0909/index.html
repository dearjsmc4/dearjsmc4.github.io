개발환경 셋팅

툴 체인: 내게 필요한 여러가지 개발환경과 도구를 제공해준다.
-> 나는 Create React App 을 사용

우선 npm 을 다운로드 한 후에 Create React App 설치

(버전 확인은 $ npm -v)
(버전 업데이트는 $ npm install npm@latest -g)

앱을 관리할 디렉토리를 만들고 cd 로 이동한 뒤

해당 디렉토리 안에 개발 환경이 셋팅된다.

실무에서는 npx 를 쓰기를 권장.
